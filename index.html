Highlighted Projects

Potluck Planner | Front-End Developer

Potluck Planner was a week long project aimed to be an easy to use "event planning" app that allowed you to create, plan, connect and share Potluck Parties with your friends. The Back-End was built using Node.js and SQL while the Front-End side was built in JavaScript using the React Hooks and Redux frameworks while also using libraries such as Bootstrap, Semantic-UI, Styled-Components, Axios, Yup, Formik and JQuery. This was a multi team project that required consistent communication between the teams and the ability to show strong leadership and resourcefulness to reach our goals.

  • Potluck Planner

Miracle Messages | Web Developer

Myself and a team of other developers from Lambda School partnered up with Miracle Messages to work on a new upcoming "Get Involved" application that will be added to the Miracle Messages main page. I spearheaded the Front-End UI by building around the Mapbox application with React and Redux, while the Back-End was built from the ground up using Node.js and SQL. The application is hosted on Heroku and is still under continual refinement as new Developers join the Miracle Messages team.

  • Miracle Messages

Guidr Responsive Landing Page | Front-End Developer

This project is a template for the Guidr App's landing page built almost solely with HTML and CSS. The landing page is meant to be a responsive first introduction to the Guidr App which was a project made for back country Guides that allowed them to advertise their own trips and journeys by updating, personalizing, and even privatizing their top trips and experiences. The Front-End was built using mainly HTML and CSS, some base JavaScript, and is very responsive.

My Work Model

6 Keys for Success

Communication

Communication is key to any project. It helps to keep everyone on the same page with clear goals and the teams vision in mind. Good communication skills are also a very vital component of anyone aspiring to become a great leader or programmer.

Repetition

Learning isn't a one and done subject. It's being able to do it over and over again with accuracy, focus, and resolve. A never-give-up attitude will grant you the ability to master it.

Big Picture

It's always important to remember the bigger picture when you find yourself struggling on the little things. Nothing worth while ever comes easy, that's why having the end goal in mind is important.

Work Ethic

An honest work ethic is derived from having authenticity in work product and educating and applying yourself. Being able to learn from your failures and request help when needed is foundational to this process.

Equipping Yourself

Being prepared and equipped for difficult times means staying innovative to better position yourself for success. Wisely fine tuning your focus will help you be prepared for the challenges ahead.

Success

There will be both success and failure in life. Learning from failures, equipping yourself through repetition of honest work, and staying focused on the big picture, will ensure your ability to master and communicate success.
